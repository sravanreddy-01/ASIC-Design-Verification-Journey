# Day 1 - Number Systems

## Topics Covered
- Decimal Number System
- Binary Number System
- Octal Number System
- Hexadecimal Number System
- Number Conversions

## What I Learned

Decimal uses base 10.

Binary uses base 2 and contains only 0 and 1.

Octal uses base 8 and contains digits 0 to 7.

Hexadecimal uses base 16 and contains digits 0 to 9 and A to F.

## Examples

45₁₀ = 101101₂

11111111₂ = 255₁₀

7A₁₆ = 01111010₂

377₈ = 255₁₀
