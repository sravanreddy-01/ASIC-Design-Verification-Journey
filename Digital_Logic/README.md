# Digital Logic

This folder contains my study notes, practice exercises, and learning progress in Digital Electronics and Digital Logic Design.

## Topics Covered

* Number Systems
* Signed and Unsigned Representations
* 1's Complement and 2's Complement
* Binary Arithmetic
* BCD Code
* Gray Code
* Logic Gates
* CMOS Logic
* Boolean Algebra
* Universal Gates
* SOP and POS Forms
* Karnaugh Maps (K-Maps)
* Combinational Circuits
* Half Adders
* Full Adders
* Ripple Carry Adders
* Multiplexers (MUX)
* Shannon Expansion Theorem

## Learning Objective

The goal of this section is to build a strong foundation in Digital Logic Design, which serves as the basis for Verilog, RTL Design, ASIC Design, and Verification.
